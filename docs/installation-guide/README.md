# Guia de Instalação

Esta seção reúne os guias de instalação e configuração para os sistemas operacionais suportados pelo projeto.

## Guias Disponíveis

| Sistema Operacional   | Descrição                                                                                                                     |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------    |
| [Arch Linux](arch/arch.md) | Instalação nativa no ArchLinux, configuração das ferramentas e programação da FPGA.                                      |
| [Ubuntu](ubuntu/ubuntu.md)   | Instalação nativa no Ubuntu, configuração das ferramentas e programação da FPGA.                                       |
| [Fedora](fedora/fedora.md)   | Instalação nativa no Fedora, configuração das ferramentas e programação da FPGA.                                       |
| [WSL](wsl/wsl.md)         | Configuração do Windows Subsystem for Linux (WSL 2), compartilhamento USB com usbipd-win e programação da FPGA pelo Linux.|
| [Windows](windows/windows.md) | Instalação do OSS CAD Suite, configuração do ambiente e gravação da FPGA diretamente pelo Windows.                    |

A versão do **[oss-cad-suite](https://github.com/yosyshq/oss-cad-suite-build)** para cada script está definida para: 
```bash
RELEASE="2026-06-19"
ARCHIVE="oss-cad-suite-linux-x64-20260619.tgz"
```
Caso queira uma versão mais recente, altere.

**obs**: Para melhor experiência use o linux $;)$.

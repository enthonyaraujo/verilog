# Guia de Instalação

Esta seção reúne os guias de instalação e configuração para os sistemas operacionais suportados pelo projeto.

## Guias Disponíveis

| Sistema Operacional   | Descrição                                                                                                                  |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| [Arch Linux](arch.md) | Instalação do OSS CAD Suite, configuração do ambiente e gravação da FPGA diretamente pelo Windows.                         |
| [Ubuntu](ubuntu.md)   | Instalação nativa no Ubuntu, configuração das ferramentas e programação da FPGA.                                           |
| [WSL](wsl.md)         | Configuração do Windows Subsystem for Linux (WSL 2), compartilhamento USB com usbipd-win e programação da FPGA pelo Linux. |
| [Windows](windows.md) | Instalação e configuração do ambiente de desenvolvimento em Arch Linux.                                                    |

**obs**: Para melhor experiência use o linux $;)$.
